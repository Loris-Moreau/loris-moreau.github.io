Some of my projects:

Grey Wolf Engine

A 3D game engine built from scratch in C++ with an integrated editor, targeting shooters, exploration games, management sims, and narrative titles.

Goals: Build a personal engine flexible enough for games like Doom, Outer Wilds, Factorio, and Mirrors Edge, while learning every layer of the engine pipeline.

GitHub Repository

3D Engine (C++)

A 3D game engine built in roughly 2 months, then extended to ship a Descent-inspired game using the same codebase.

Goals: Learn about 3D engines, rendering pipeline, and core game systems.

GitHub Repository

Raytracer, Compute Shaders

Built a full path tracer following the Ray Tracing in One Week series, then pushed render times from 5 hours down to seconds using Compute Shaders, SIMD, and multi-threading.

Goals: Understand rendering fundamentals, then systematically profile and optimize, covering parallel computing, SIMD intrinsics, and GPU compute pipelines.

GitHub Repository Read the blog post

UE5 AI System

A behaviour-tree driven NPC AI in Unreal Engine 5 with sight & hearing detection, dynamic patrol routes, and adaptive difficulty.

Goals: Make an AI that reacts believably to the player while staying fun to fight, striking the right balance between challenge and fairness.

GitHub Repository

Conway's Game of Life

A C++ implementation using a sparse data structure for an infinite grid, built with GLFW, Glad, ImGui, and OpenGL.

Goals: Learn ImGui and OpenGL, and explore cellular automata, always fascinated that such simple rules can produce a Turing complete system.

GitHub Repository

Houdini Tools

Procedural tools and simulations built in Houdini 19 & 20, exploring VEX, SOPs, and DOPs to create game ready assets and effects.

Goals: Learn procedural content generation and build reusable tools that I can bring directly into my games.

Related blog posts

UE5 VFX & Shaders

Shaders, materials, and visual effects in Unreal Engine 5 using Niagara and custom HLSL, covering particle systems, post-process effects, and material functions.

Goals: Experiment with real-time VFX and build a personal shader library usable across game projects.

Related blog posts

Unity VFX & Shaders

Shaders and visual effects in Unity testing the Shader Graph and custom HLSL pipeline.

Goals: Build reusable shaders and compare Unity's pipeline to Unreal's material system.

GitHub Repository

Windows Adventure (Godot)

A platformer where OS windows appear along your path, some help, others just block the view. A quick prototype to experiment with Godot's window management API as a game mechanic.

Goals: Learn Godot and explore unconventional UI as a gameplay element.
